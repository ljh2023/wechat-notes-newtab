从微信读书同步

通过微信读书官方 API 自动获取你的所有笔记，无需手动导出 JSON。

1 前往微信读书获取 API Key 打开 →
2 将 API Key 粘贴到下方
3 点击同步，自动获取所有笔记

准备中...

💡 API Key 仅保存在本地浏览器中，不会上传到任何服务器。

导入笔记

将微信读书笔记以 JSON 格式导入。支持直接粘贴或上传文件。

点击选择 JSON 文件上传，或拖拽到此处

或
查看示例格式 
[
  {
    "id": "1690000000001",
    "content": "人的认知边界决定了其解决问题的上限。",
    "book": "认知觉醒",
    "author": "周岭",
    "chapter": "第四章",
    "createTime": "2024-01-15"
  },
  {
    "id": "1690000000002",
    "content": "真正的行动力来自于清晰的目标和明确的路径。",
    "book": "认知觉醒",
    "author": "周岭",
    "chapter": "第五章"
  }
]

必需字段：id、content。
推荐字段：book、author、chapter、createTime。
id 可以传时间戳或 UUID，用于去重。
